summary: Implementé control automatizado basado en visión para manos protésicas usando procesadores FPGA, con cámara RGB interfaceada a Zynq UltraScale+ MPSoC con modelos de aprendizaje profundo para detección de objetos.
stack:
  - Zynq UltraScale+ MPSoC
  - Diseño FPGA
  - Visión por computadora
  - Aprendizaje profundo
  - Pipeline de procesamiento de imágenes
  - Interfaz de cámara RGB
  - Diseño de circuitos digitales
impact:
  - Abordé limitaciones de alto consumo de energía en robótica sofisticada mediante implementación FPGA eficiente en energía.
  - Desarrollé pipeline avanzado de procesamiento de imágenes desde cero para aplicaciones de mano protésica.
  - Creé repositorio disponible públicamente con documentación exhaustiva de implementación.
  - Habilité sensores de visión para detectar artificialmente objetos desde posición de muñeca para control sofisticado de agarre.
